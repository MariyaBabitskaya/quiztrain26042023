jQuery – это библиотека JavaScript
Предназначена для абстрагирования, выравнивания, исправления и
упрощения скриптинга при работе с узлами HTML-элементов в браузере
или для работы в браузере без графического интерфейса.

Все это оборачивается в более простой и менее корявый API,
нежели нативный API DOM – вот вам и библиотека jQuery.

При помощи jQuery совершенно тривиально решаются задачи вроде
«визуально скрыть второй элемент h2 в документе .html.
jQuery('h2:eq(1)').hide();

метод хайд собственно скрывает элемент

код в более обычном варианте:
document.querySelectorAll('h2')[1].style.setProperty('display','none');
